Island Levels Mechanism

This page will explain how the island levels works.

Calculation

  • Value of one island level is 100
  • Due to various requests, there is no death penalty
  • Your island level is updated every time you join the server
    You can manually update it with /is levels
  • Limits
    • Cobblestone10000
    • Stone5000
    • Wool1000
    • Netherrack1000
    • Logs1000

Block Values

Many blocks in-game worth a certain value of points, the total island points is the sum of all the island's blocks's values.
